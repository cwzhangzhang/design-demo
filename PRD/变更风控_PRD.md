# 变更风控

高速网络变更管理系统，用户在此完成变更动作。

## 【变更风控】产品页面功能描述及交互描述

### 说明
- 高速网络变更管理系统，用户在此完成变更动作。

### 列表页
- 搜索条件：
  - `变更标题`
  - `等级`
  - `状态`
  - `变更模版`
  - `变更人`
  - `变更时间`
- 按钮：
  - `发起变更`：点击打开 `发起变更（抽屉）`。
- 列表展示：
  - 字段：`变更标题`、`等级`、`状态`、`审批单`、`变更模版`、`变更人`、`变更时间`、`执行单`、`进度`
- 行内操作：
  - `查看详情`：进入 `详情页` 查看变更详细内容。

---

### 发起变更（抽屉）
- 打开方式：点击列表页 `发起变更` 按钮。
- 按钮：
  - `提交`：提交变更申请。
  - `取消`：关闭抽屉。
- 表单字段：
  - `变更基本信息`：
    - `变更标题`
    - `变更对象`
    - `变更时间`
    - `变更场景标识`
  - `变更等级与影响`：
    - `变更等级`
    - `变更影响范围`
    - `回滚方案`
  - `变更内容`：
    - `变更模版`：
      - `1. RDMA Agent 配置变更（各种配置、CC、网卡配置等）`：需选择 `List 下发的命令（单选）`。
      - `2. RDMA 配置检查（各种配置当前状态的获取，针对未知新发现的问题的配置扫描）`：需选择 `List 下发的命令（单选）`。
      - `3. RDMA Agent 版本升级（日常迭代）`：需选择 `版本选择`。
      - `4. 每一个模版回显灰度策略和观测与风控规则`
- 行为：按层级填写变更申请信息。
- 返回：点击 `取消` 或点击抽屉外部返回列表。

---

### 详情页
- 说明：查看变更的详细执行进度与报告。
- 基本信息：
  - 字段：`变更名称`、`申请人`、`整体状态`、`变更模版`、`创建时间`、`变更主机数量`、`成功机器数量`、`失败机器数量`
- 总任务执行进度图：
  - `进度步骤条`
  - `进度详情`：
    - 列表展示：`阶段名称`、`阶段状态`、`机器灰度比例`、`机器数量`、`变更观测`
- 页面操作：
  - `变更观测监控`（按钮）：抽屉展开展示每一阶段的监控指标图。
  - `观测确认成功`（按钮）：每一个步骤点击观测确认成功之后才可以进行下一步
  - `观测结果异常`（按钮）：点击观测结果异常，则停止本次变更
- 执行报告：
  - 列表展示：
    - `主机IP`（支持搜索查询）
    - `状态`（支持筛选）
    - `汇报时间`
